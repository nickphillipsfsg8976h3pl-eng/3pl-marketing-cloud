The 2025 school year is almost here! Let’s kick things off by completing the rollover process and ensuring that all your students are in the right classes for the year.
 

Rollover is even easier this year! Simply download this empty spreadsheet, fill it in, and send it back to me at nish.tomczyk@3plearning.com and we will take care of the rest.

 

How to complete your Reading Eggs Rollover

  1. Download this empty spreadsheet.
  2. Fill it in with your updated class details.
  3. Send it back to me at nish.tomczyk@3plearning.com, and we will take care of the rest!

Important: Please do not make any changes to current students’ names on the spreadsheet, as this could result in duplicate accounts getting created and the loss of previous data.

 

For data security purposes, we ask that you encrypt the file with a password in a separate email.

  1. Select 'File' → 'Info'.
  2. Select the 'Protect Workbook' box and choose 'Encrypt with Password'.
  3. Enter a password in the Password box, and select 'OK'.
  4. Confirm the password in the 'Re-enter Password' box, and then select 'OK'.
