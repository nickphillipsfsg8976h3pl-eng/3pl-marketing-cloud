To create a new class simply:
  1. Log in to your Reading Eggs Teacher Dashboard
  2. Select 'Manage Classes'
  3. Enter the class name
  4. Click 'Go'
  5. To add students/pupils, click ‘Manage Students/Pupils’, then ‘Move Students/Pupils’ into your class


To edit an existing class:
  1. Log in to your Reading Eggs Teacher Dashboard
  2. Select 'Manage Classes'
  3. Click the edit icon beside the class
  4. Make the necessary changes
  5. Click 'Save changes'
