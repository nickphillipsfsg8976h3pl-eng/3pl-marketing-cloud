Resetting the Placement Test
 

At any time, you can get your students to redo the Placement Test. This is great way to make sure they are still working at the right level.
 

1. Click Manage Students.

 

2. Select a student or students.
 

3. Click the Edit Progress button.
 

4. Select Placement Test.
