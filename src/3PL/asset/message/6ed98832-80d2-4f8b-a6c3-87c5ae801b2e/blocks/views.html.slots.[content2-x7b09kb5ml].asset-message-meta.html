Unlock tailored learning: Create student ability groups

Mathletics allows you to create learning groups based on your students' current abilities which makes assigning differentiated content easier than ever.

How to create a group:

  1. Log into your Mathletics Teacher Console
  2. Click 'Manage Students
  3. Select 'Groups', then 'New Group'
  4. Fill in the new group information
  5. Add students to their ability group by using the dropdown and selecting the appropriate group.
