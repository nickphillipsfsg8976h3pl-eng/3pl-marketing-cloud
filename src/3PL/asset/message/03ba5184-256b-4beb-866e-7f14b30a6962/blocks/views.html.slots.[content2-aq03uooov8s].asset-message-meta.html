üïÆ How to Assign/Reassign the Mathseeds Placement Test
The Mathseeds Placement test is automatically assigned to students when they first log into Mathseeds. To reset the Placement Test follow these steps:

  1. Sign into Mathseeds
  2. Select Mathseeds from the lefthand menu
  3. Click Management
  4. Select Manage Progress from the list
  5. Find the name of the student that you want to reset the placement test for, and select the tickbox on the lefthand side of their name
  6. Click Edit progress and choose Placement Test
  7. Click the Update button
