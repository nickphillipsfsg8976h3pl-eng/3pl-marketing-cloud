How to complete your Reading Eggs Rollover

For data security purposes, we ask that you encrypt the file with a password in a separate email.

  1. Select 'File' → 'Info'
  2. Select the 'Protect Workbook' box and choose 'Encrypt with Password'
  3. Enter a password in the Password box, and select 'OK'
  4. Confirm the password in the 'Re-enter Password' box, and then select 'OK'.
