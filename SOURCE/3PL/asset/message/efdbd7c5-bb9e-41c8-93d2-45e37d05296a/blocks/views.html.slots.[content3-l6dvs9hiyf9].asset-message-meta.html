Reading Eggs contains 120 lessons spread out over 12 learning maps and Reading Eggspress has 220 lessons over 44 maps. Creating and managing assignments works the same in both Reading Eggs and Reading Eggpress.
 
  1. Click Manage Assignments.
  2. Use the check boxes to select individual student(s) or Select all for a group assignment.
  3. Select the start date and the due date.
  4. Choose the Type of Activity you want to assign.
  5. Review the details and click Create this assignment.
