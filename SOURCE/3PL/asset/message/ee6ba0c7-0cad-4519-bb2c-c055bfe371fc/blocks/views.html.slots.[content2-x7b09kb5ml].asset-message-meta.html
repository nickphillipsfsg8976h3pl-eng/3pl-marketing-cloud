To add a student/pupil simply follow these steps:

  1. Log in to your Mathseeds Teacher Dashboard
  2. Click on 'Manage Students/Pupils’
  3. Enter the student/pupil details
  4. Click 'Go'
  5. If you need to add multiple students/pupils, use the 'Upload a CSV' option.

To edit existing student/pupil details:
  1. Log in to your Mathseeds Teacher Dashboard
  2. Navigate to 'Manage Students/Pupils’
  3. Click the edit icon next to the student's name
  4. Make any necessary edits
  5. Click 'Save changes'
