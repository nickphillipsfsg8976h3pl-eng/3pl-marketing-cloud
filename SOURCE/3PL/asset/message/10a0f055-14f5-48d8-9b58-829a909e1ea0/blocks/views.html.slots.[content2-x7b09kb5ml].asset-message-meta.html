To add a student/pupil simply follow these steps:
  1. Log in to your Reading Eggs Teacher Dashboard
  2. Click on 'Manage Students/Pupils’
  3. Enter the student/pupil details
  4. Click 'Go'
  5. If you need to add multiple students/pupils, use the 'Upload a CSV' option.
 



To add a student/pupil simply follow these steps:
  1. Log in to your Reading Eggs Teacher Dashboard
  2. Click 'Assignment'
  3. Choose 'Create Assignment'
  4. Select the students you want to take the test
  5. Set the start and due dates
  6. Opt for the ‘Driving Test’ assignment type
  7. Click Create this assignment
