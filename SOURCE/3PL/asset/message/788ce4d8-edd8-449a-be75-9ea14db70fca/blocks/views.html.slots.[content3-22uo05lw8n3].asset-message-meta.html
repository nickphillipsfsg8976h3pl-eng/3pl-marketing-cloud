• The Achievement and Improvement report to show your class or group's progress across topics.
  • The Effort and Participation report to see stats on points, certificates and time spent online.
  • The Activity and Usage report to download data in a Excel file
  • Student reports to see Skill Quests or Activities mastery.
 
 
 
 

1. Select your class or group and then use the date range filter.
 

2. Hover over each box plot to view the highest and lowest result, and the upper and lower quartiles.

 

3. Click a box plot and scroll down to view the average results for that topic.
 

4. To see more details for each activity, select individual students or select all students, then click See results.
