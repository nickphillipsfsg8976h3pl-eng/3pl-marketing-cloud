The Mathseeds Parent Letter and Parent Pack are great ways to involve parents in their child's progress and foster extra practice!

The Mathseeds Parent Letter is a handy PDF that contains login information and how to create a parent account, for parents to access Mathseeds.

Print the parent letter by following these steps:

  1. Log into your Mathseeds Teacher Dashboard and toggle to your 3 Essentials page.
  2. Navigate to Students
  3. Selecting the students you want to pring a parent letter for
    (Pro tip: Select All for a quick way to print parent letters for your entire class)
  4. Click Parent Letters from the top menu
  5. Click Mathseeds from the available options

You can find additional resources in the Mathseeds Knowledge Base, like the Mathseeds Parent Pack
