To add a student simply follow these steps:

  1. Log in to your Reading Eggs Teacher Dashboard
  2. Click on 'Manage Students’
  3. Enter the student details
  4. Click 'Go'
  5. If you need to add multiple students, use the 'Upload a CSV' option.
